# Stock Price Volatility Prediction using Financial News

## Project Overview
This project analyzes the relationship between financial news and stock price movements. 
The pipeline involves collecting financial news from RSS feeds, cleaning and standardizing the data, 
collecting stock price data, and intelligently merging both datasets for downstream analysis.

## Tasks Completed
- Multi-source financial news collection using RSS feeds
- XML structure understanding
- News data cleaning and standardization
- Stock price data collection using yfinance
- Market calendar awareness
- Intelligent data merging
